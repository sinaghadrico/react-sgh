# react-strix

React mega starter

- next.js
- styled-components
- typescript
- docker
- express
- storybook
- atomic-design
- resolve-import
- eslint
- jest

base on [orbit.kiwi](https://orbit.kiwi)

## showcase

- [demo](https://react-sgh.straxico.now.sh)
- [storybook](https://straxico.sinaghadrico.io/react-sgh/)

## Installation

- `git clone https://github.com/sinaghadrico/react-sgh.git`
- `cd react-sgh`
- `npm install`
- `npm start`
